## Cmd Variables
--------
### TXT_TO_CLIP 
Copy text to clipboard and launch gpt35 site
### REMOVE_TEXT
Remove request from list

### bookmarkListName 
Input or select bookmark list file name
- Prefix must be "book" 
	ex) "bookSites"
